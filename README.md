## 📄 `word-ex/README.md`

# 🎮 WORD STAGE - 全文ハック型・英単語特訓システム -

GitHub Pagesで爆速で動く、音ゲー風UIを意識したサイバーパンクデザインの英単語タイピング特訓アプリです。
学校の単語テストの範囲（JSON）を動的に切り替え、リロードなしでノンストップ周回プレイが可能です。

---

## ⚡ 主な機能 (Features)

* **マルチステージ・ハック（JSON動的切り替え）**
    * `json/menu.json`（目次ファイル）を更新するだけで、HTMLやJSのコードを一切弄ることなく、新しいテスト範囲のステージを無限に自動追加できます。
* **自由な穴あき問題形式 (Custom Sentence Input)**
    * お友達からの熱い要望により、文章の中に `[ _ _ _ ]` や カッコなどの文字列表現をそのまま残して出題可能です。打ち込む正解はターゲットとなる単語のみなので、実践的なテスト対策ができます。
* **ノンストップ周回システム (Non-reload Loop)**
    * リザルト画面からタイトルへの帰還時、ブラウザの再読み込み（リロード）を挟まないため、選択したステージ状態を維持したまま、SSSランクを目指して何周でも爆速でローテーション特訓が可能です。
* **お気に入りマネージャー (Favorite Management)**
    * プレイ中に★をトグルできるだけでなく、「📋 WORD LIST」タブからその範囲の単語を一覧で確認し、その場でポチポチとお気に入りのON/OFFを一括管理できます。

## 🛠 新しいテスト範囲（ステージ）の追加手順

今後、新しい単語テストの範囲を追加したい場合は、以下の**2ステップ**だけで完全自動反映されます。**HTMLやJavaScriptを書き換える必要は一切ありません。**

### 1. `json/` フォルダ内に新しい問題データを作成

例えば `json/words_0707.json` を作成し、以下の形式でデータを記述します。

```json
[
  { 
    "word": "one of my best friends", 
    "sentence": "He is [ _ _ _  _ _  _ _  _ _ _ _  _ _ _ _ _ _ _ ].", 
    "meaning": "彼は私の親友の一人だ。" 
  }
]

```

* `word`: ユーザーが実際にキーボードで打ち込む**正解の文字列**です。
* `sentence`: 画面に出題される文章です（空欄表記などを自由にデザインできます）。
* `meaning`: 日本語の翻訳やヒントです。

### 2. `json/menu.json` に追記

作成したファイル名と、画面に表示したいステージのタイトルを目次ファイル（`menu.json`）の末尾に追記します。

```json
[
  { "file": "json/words_0616.json", "title": "📅 6月16日 単語テスト" },
  { "file": "json/words_0623.json", "title": "📅 6月23日 単語テスト" },
  { "file": "json/words_important.json", "title": "🔥 期末向け重要単語" },
  { "file": "json/words_0707.json", "title": "🌌 7月7日 七夕特訓テスト" }
]

```

## 🚀 開発の美学 (Development Aesthetics)

このプロジェクトは、「DRY原則（Don't Repeat Yourself：同じコードを何度も書かない）」**と**「ユーザー体験のテンポ感」を極限まで追求して開発されました。
リロードなしの画面遷移、LocalStorageを使用したシームレスなお気に入り同期など、エンジニアとしてのこだわりが詰まった単語帳システムです。
